Interactive Adventure Game

  • Tech Stack: Java, Java Swing, MVC
  • Description: The world for our game consists of a dungeon, a network of tunnels and caves that are interconnected so that player can explore the entire world by traveling from cave to cave through the tunnels that connect them. The player has to reach the end of the game without getting killed by Monster. Otyughs are smelly monsters which reside in the dungeon. Players can detect two levels of smell:a less pungent smell can be detected when there is a single Otyugh 2 positions from the player's current location detecting a more pungent smell either means that there is a single Otyugh 1 position from the player's current location or that there are multiple Otyughs within 2 positions from the player's current location. Players can slay Otyughs using arrows it takes two hits to kill an Otyugh. There is a flying monster in the game which the player has to fight with. If the players health drops below 0 then the player loses the game. There is also a theif in the game which the player has to avoid who can steal all the treasures. Finally the player has to avoid one pit which can decrease the player by half if the player falls on to it.
